Notes about entering the value for "city":

  • Be cautious with spelling - there's no autocorrect. If it's misspelled, then the submission won't go through.
  • If state/province is not provided, it will default to the largest city of that name.
    • Entering "baltimore" (no state) will return "Baltimore, MD"
    • Entering "baltimore, md" will return "Baltimore, MD"
    • Entering "baltimore, oh" will return "Baltimore, Ohio" (yes, that's a real place)
  • The city-validation checks against open-source mapping services, so if you enter "asdf", you might find somebody has created it as a place in some mapping service.

Note about date entry:

The year value is irrelevant. I chose a calendar date-entry form to avoid users entering an invalid date (Nov 31st, etc.) which would cause things to not work. Once you a pick a date, I just ignore the year value.
